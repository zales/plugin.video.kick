KICK.com Kodi Add-on

Latest: v2.6.10  •  All releases

1 — Add repository source (one-time)

  1. Kodi → Settings → File Manager → Add source
  2. URL: https://kodi.zales.dev   Name: KICK repo

2 — Install the repository

  1. Kodi → Add-ons → Install from zip file → KICK repo
  2. Select repository.zales.kick → repository.zales.kick-1.0.0.zip

3 — Install the add-on (auto-updates)

  1. Kodi → Add-ons → Install from repository → KICK.com Repository
  2. Select Video add-ons → KICK.com → Install

Alternative — direct install (no auto-updates)

  1. Download: plugin.zip  https://kodi.zales.dev/plugin.zip
  2. Kodi → Add-ons → Install from zip file → select the downloaded zip
